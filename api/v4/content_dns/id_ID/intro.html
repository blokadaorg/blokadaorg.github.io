Selamat datang dan terima kasih telah menggunakan Blokada DNS!

Blokada adalah aplikasi pemblokir iklan gratis, open source, dan menghargai privasi Anda. Apabila Anda memutuskan untuk menggunakan Blokada, maka Anda dianggap menyetujui Ketentuan Layanan dan Kebijakan Privasi, yang detailnya bisa Anda lihat di sini kapan saja.

Kami ingin membantu Anda dengan langkah pertama Anda:

  1. Saat ini Anda sedang menggunakan DNS server bawaan (yang ditentukan oleh provider seluler atau router WiFi Anda). Anda dapat mengubahnya dengan menekan ikon DNS Bawaan.
  2. Blokada dikonfigurasikan untuk mulai aktif ketika perangkat Anda dinyalakan. Anda dapat mematikan pengaturan ini dengan memilih Mulai Saat Boot.
  3. Blokada hanya akan aktif ketika Anda terhubung ke internet. Anda dapat mengubahnya dengan menonaktifkan Hanya Online.
  4. Di Whitelist Anda dapat menentukan aplikasi mana saja yang tidak diperbolehkan memakai server DNS Anda. Aplikasi yang masuk Whitelist akan menggunakan server DNS bawaan perangkat Anda.
  5. Tekan 3 titik di bagian kanan atas untuk menemukan lebih banyak opsi dan menu Bantuan di mana Anda dapat memperoleh informasi mendetail tentang berbagai fitur yang ditawarkan oleh Blokada.
