Benvenuti e grazie per aver utilizzato Blokada DNS!

Blokada è gratuita, open source e rispetta la tua privacy. Utilizzandola, accetti i Termini di Servizio e l'Informativa sulla Privacy, che puoi sempre trovare qui.

Saremmo lieti di aiutarti con i primi passi:

  1. Per il momento sono in uso i DNS predefiniti. Puoi cambiarli toccando l'icona DNS predefiniti.
  2. Blokada è impostata per l'avvio all'accensione del telefono. È possibile modificare questa impostazione disabilitando l'opzione Avvia al boot.
  3. Blokada entrerà in funzione solo quando si è connessi a internet. È possibile modificare questa impostazione disabilitando l'opzione Solo online.
  4. Nella Whitelist puoi impostare le app che non vuoi utilizzino i tuoi server DNS. Queste utilizzeranno i server DNS predefiniti del dispositivo.
  5. Dall'angolo in alto a destra puoi trovare altre opzioni e il menù di Aiuto dove puoi ottenere informazioni dettagliate sulle diverse opzioni di Blokada.
