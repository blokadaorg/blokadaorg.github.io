Blokada บล็อกโฆษณาน้อยลง

ถ้าคุณสงสัยว่า Blokada บล็อกโฆษณาน้อยลงกว่าก่อนหน้านี้ โปรดลองฟื้นฟูรายการของคุณ To do it, tap on Blacklist, then in the top right corner on the equaliser, choose Refresh filters and tap on Continue. Usually it happens because new servers are added to the existing ones that are not (yet) identified and added to the blacklists, or the new list is not downloaded to your phone, therefore they are not (yet) blocked.
Alternatively you can enable more host lists or add more restrictive filters in the Blacklist menu.

Please note that Blokada uses 3rd party lists, therefore if the app itself worked earlier and blocked ads, without changes it should behave the same way.

แอพไม่ทำงานตามที่หวังไว้

If you can see that an app doesn't work as expected, doesn't load its content or something similar misbehaviour is experienced, please go to the main screen, choose Whitelist, at the bottom of the screen tap on the + sign, choose App and type its name. Tap on Save at the end. With this, Blokada won't filter the URLs in this app.

Please note that Blokada 3.4 and earlier releases had a bug which caused that a freshly installed app wasn't recognized in Blokada, therefore they couldn't be set on the Whitelist. This bug is fixed in version 3.5 and newer.

Ads in YouTube aren't blocked!

Blocking ads on YT is very hard because Google uses encryption and randomly generated hosts to serve the ads. Therefore, it's much easier to block ads within the YT app as with an external tool (like Blokada). NewPipe and YouTube Vanced are featureful, great alternatives of the official YouTube application, please check which fits better for you.

ฉันไม่สามารถเริ่มใช้งาน Blokada หลังติดตั้งเสร็จ

มีสองสาเหตุที่พบบ่อยๆในสถาณการณ์นี้ อย่างแรกลองเช็คดูว่า ถ้าคุณมีการเชื่อมต่อ VPN อยู่ และถ้าตัวเลือก VPN แบบเปิดตลอด นั้นถูกเปิดอยู่ ให้ทำดังนี้ ไปที่การตั้งค่าของมือถือคุณแล้วค้นหา VPN ในส่วนของระบบไร้สายและเครือข่าย แตะที่ไอคอนรูปเฟืองของ VPN ที่มีอยู่ เลือกปิดมันซะ ตัดการเชื่อมต่อจากการเชื่อมต่อ VPN ที่ใช้มีอยู่ และลองปิดและเปิดแอพ Blokada ใหม่อีกครั้ง

If you don’t have other VPN connection, check in Blokada if Watchdog is enabled in the Advanced Settings, and if yes, check if you are connected to the internet either with wifi or with mobile data. Blokada won’t start if the Watchdog is enabled and you don’t have an active internet connection.

ฉันไม่สามารถนำเข้ารายการ EasyList/AdBlock Plus มาใช้ใน Blokada ได้

If you try to import a list (for example EasyList) that works very well with your AdBlock Plus extension on your PC or notebook, Blokada will report an error message. It is because Blokada uses other method to block ads. While ABP blocks and hides elements or parts of the website, Blokada cannot do it because it's not a plugin in the browser, instead it redirects the accesses to the ad-server to a non-routable address (0.0.0.0), therefore it needs a hostname-IP list.

Blokada ทำงานอย่างไร?

Blokada เป็นชั้นกั้นบางๆระหว่างตัวการเชื่อมต่อเน็ตเวิร์ค (WiFi หรือ เน็ตมือถือ) ของคุณและแอพลิเคชั่นที่คุณใช้ To be able to filter every outgoing traffic, it generates a local (no 3rd party service in use) VPN connection and simply blocks any requests towards the hosts defined in the blacklist filters. การบล็อกนี้จะทำการเปลี่ยนเส้นทางของโฆษณา ที่อยู่ IP ของเว็บไซต์ที่เซิร์ฟเวอร์ของโฆษณาจะถูกแก้ไขโดยโทรศัพท์ด้วยการระบุ meta-address ที่ไม่สามารถใช้งานได้ สำหรับ 0.0.0.0 ที่ไม่มีบริการเว็บทำงานอยู่เลย ดังนั้นจะไม่มีพวกโฆษณาโชว์ขึ้นมา ด้วยสิ่งนี้ Blokada ประหยัดทั้งแพ็คเกจเน็ตคุฯและอายุการใช้งานแบตเตอร์รี่ ตัวกรองจะได้รับอัพเดตทันทีและประกอบไปด้วยโฮสต์ที่รู้จักกันว่าบล็อกโฆษณา, การติดตาม, สแปม, ไวรัส, และสิ่งที่ไม่ต้องการอื่นๆหรือเนื้อหาอันตราย คุณสามารถตั้งค่า Blokada ได้ด้วยตัวกรองปรับแต่งโดยคุณ และรายการยกเว้นเพื่อให้บริการที่คุณชอบจะไม่ต้องโดนบล็อก และคุณจะไม่ต้องพบกับประสบการณ์แย่ๆ

ฉันสามารถใช้ VPN ตัวอื่นระหว่างเปิดใช้งาน Blokada ได้หรือไม?

Unfortunately no, it is a system limitation; you can have only one active VPN connection at a time. If you join to an other service, Blokada will exit.

ตัวเลือกใน Blokada มีอะไรบ้าง?

As you surely saw it, Blokada has some options on the main screen that are not detailed at the first start. Please let us do it now. You can see the following:

  • Blacklist: this is the lists of host files containing the servers to block. Please note that using multiple lists can have effect on network and system speed.
  • Whitelist: the whitelist is the exception list, here you can define hosts that should not be blocked or apps where Blokada won’t filter URLs. Blokada is installed with several whitelisted apps by default.
  • Keep alive: this option is to ensure Blokada is not paused or stopped by the system. Please note your system can overwrite it, make sure your settings will let Blokada run in the background.
  • Notifications: when enabled, you will be notified when an ad is blocked. It is on by default, but if you get a lot of notifications, turn it off.
  • Watchdog: when enabled, Blokada will exit if there is no active connection, and start again whenever connection comes back. If disabled, Blokada will always run in the background (even if no network is available). Please note this option may not work on every device.
  • Start on boot: when enabled, Blokada will start after the system boots up. Please note there are system level configurations for this option, therefore it cannot be guaranteed Blokada can start on boot. In this case, please check your phone’s settings. The default configuration is enabled, if you want to start Blokada manually, disable this option.
  • DNS: if you are suspecting that some websites or services are blocked for you, or you are presuming that your internet provider collects information about your intenet usage, choose a secure DNS machine from the list. By default, Blokada uses the same DNS server/s as your system.

Not configurable things:

  • Up to date: if you see it, you are running the most recent version of Blokada. Keep the app up-to-date because it has less bugs and more features!
  • # ads blocked: the number of unwanted content that Blokada blocked since it is started for the first time
  • # hosts: number of servers that serves advertisements, malware and other unwanted content

ฉันสามารถทำอะไรได้บ้างถ้ายังเห็นโฆษณาอยู่บ้าง?

Some ads will come through with any ad blocker on Android. In particular YouTube ads or some fullscreen popup ads are difficult to block. If you think more ads are coming through than before, try these steps:

  • Check if the VPN key icon is present in the status bar at the top. If not, start Blokada.
  • Open Blokada and check if it is active and not reporting any errors.
  • Check if the hosts counter is not equal to zero. If it is, select some blacklist filters so that ads can be blocked. If the counter is still zero, grant storage access permission in the System Settings - Applications - Blokada, then clear data and cache and reboot the device.
  • Make sure you don't have too many apps in the whitelist - maybe that's why you are seeing ads.
  • Try resetting your filters (go to the Blacklist screen and choose Restore Defaults from the Quick Actions menu at the top).
  • If you see ads in your browser, deactivate any kind of traffic compression or optimisation in the browser's settings.
  • Deactivate Blokada, close it (by swiping it out from the Recents screen), and turn it on again.
  • If you recently changed your settings (filters, whitelist, etc), remember it takes time before new configuration start working. Give it some time.
  • Clear app data for Blokada (in system settings), or reinstall it.
  • Restart your device.
  • Your device may be unsupported. Please check Devices Database.
  • If none of those steps helped, please report a bug (see below).

DNS คืออะไร และฉันควรเลือกอันไหน?

The acronym DNS stands for Domain Name System and this servers are responsible to resolve the domain names (for example: myexample.com) to IP addresses, what are used between machines to communicate. Your internet service provider has its own DNS servers; Blokada is configured to use the system defaults. The machines in this list should be used if you are suspecting that your ISP collects logs about your internet usage, you are afraid of some privacy risk or if some websites or services are blocked for you. There is no "best", everything depends on your preferences and location, therefore if you would like to use custom DNS servers, please read their description, search for them on the internet and read the feedbacks.

ฉันไม่สามารถติดตั้งอัพเดตได้? ฉันต้องทำยังไง?

First of all, please check if the Android version you have is 5 or newer. Android 4 and older versions are not supported in Blokada 3.0 and newer versions. To do it, go to the Settings of your phone, scroll to About phone and search for Android version

If your phone runs Andriod 5 or newer, please restart your device. It sounds strange, but sometimes it helps. If that didn't solve the issue, uninstall Blokada and install it again. It can be, that your device is unsupported. Please check Devices Database.

If none of those steps helped, please join to our Telegram chat on go.blokada.org/chat to get help, or report a bug (see below).

ฉันสามารถซ่อนการแจ้งเตือน VPN ได้ไหม?

Unfortunately no. Some devices may be displaying annoying VPN notification (a key icon) while VPN is active, and it is not possible to hide it. If you can, try updating your device to Android 8 (Oreo), since newer Android versions do not have this status report.

Blokada มองเห็นข้อมูลส่วนตัวของฉันหรือไม่?

No. Blokada does not read or tamper with your private information, it only monitors which hosts your device is making requests towards and blocks the ones matching the filters. Every time Blokada blocks anything, you will see a notification (what you can turn off). Blokada has no access to your encrypted traffic.

Blokada ใช้แบตเตอรี่เยอะหรือไม่?

No. Depending on your settings the battery impact is minimal, and on most new devices, especially on Marshmallow (Android 6) or newer, you will not notice any battery drain. However, some unlucky devices may experience increased battery drain. Refer to the Devices Database for details.

Blokada จะทำให้เครื่องของฉันช้าหรือเปล่า?

No. Depending on your settings Blokada can be quite efficient. In fact, both your device's performance and battery life may visibly improve, because Blokada saves your device from downloading and processing content you don't need or care about. However, users with some older devices may experience performance drop. Refer to the Devices Database for details.
