Velkommen og takk for at du bruker Blokada DNS!

Blokada er gratis med åpen kildekode som respekterer ditt personvern. Ved å bruke den godtar du vilkårene for bruk og personvern, som du alltid kan finne her.

Vi vil gjerne hjelpe deg med å komme i gang:

  1. Standard DNS-servere blir brukt for øyeblikket. Du kan endre det ved å trykke på Standard DNS- ikonet.
  2. Blokada er satt til å starte når telefonen slås på. Du kan slå av funksjonen ved å trykke på Start ved systemoppstart.
  3. Blokada vil kjøre bare når du er koblet til internett. Du kan endre det ved å sette Kun online til av.
  4. I Hvitelisten kan du definere hvilke applikasjoner som ikke skal bruke DNS-serverne dine. De vil bruke enhetens standard DNS-servere i stedet.
  5. Ved å trykke i øvre høyre hjørne, kan du finne flere valg samt Hjelp-menyen hvor du kan finne detaljert informasjon om de forskjellige valg du har i Blokada.
