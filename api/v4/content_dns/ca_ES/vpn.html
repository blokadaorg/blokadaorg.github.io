Per què una VPN?

Un servei VPN augmenta la seguretat i l'anonimat a Internet, ho fa xifrant la vostra comunicació i emmascarant la vostra adreça IP real.

El xifratge passa entre el client (el dispositiu que utilitzeu actualment) i el servidor VPN i, per això, els tercers no poden llegir les dades que transferiu.

Casos d'ús típics:

  • Mentre utilitzeu punts d'accés Wi-Fi oberts o gratuïts
  • Per accedir al banc o a qualsevol altre lloc web important a l'estranger
  • Per accedir a llocs web restringits a la vostra zona
