O Blokada bloqueia menos anúncios.

Se suspeitar que o Blokada bloqueia menos anúncio que anteriormente, actualize a sua lista. Para o fazer, pressione em Lista Negra, depois no canto superior direito no equalizador, seleccioneActualizar filtros e pressione em Continuar. Usually it happens because new servers are added to the existing ones that are not (yet) identified and added to the blacklists, or the new list is not downloaded to your phone, therefore they are not (yet) blocked.
Alternatively you can enable more host lists or add more restrictive filters in the Blacklist menu.

Please note that Blokada uses 3rd party lists, therefore if the app itself worked earlier and blocked ads, without changes it should behave the same way.

Uma aplicação não funciona como esperado.

Se consegue ver que uma aplicação não funciona como seria de esperar, não carrega conteúdos ou algum comportamento parecido é observado, por favor, vá ao ecrã principal, escolha Lista Branca, pressione no botão com o símbolo "+" no fundo do ecrã, escolha Aplicação e insira o nome desta. No fim, toque em Guardar. Com isto, o Blokada não irá filtrar os URLs nesta aplicação.

Please note that Blokada 3.4 and earlier releases had a bug which caused that a freshly installed app wasn't recognized in Blokada, therefore they couldn't be set on the Whitelist. This bug is fixed in version 3.5 and newer.

Quero bloquear um determinado site.

Você pode bloquear websites individuais que não estão numa lista de servidores. Para o fazer, vá a Lista Negra na página principal, pressione no simbolo + na parte de baixo do ecrã, pressione em Servidor e insira o nome do website, sem a parte do http ou www (por exemplo: omeuexemplo.com).

Anúncios no YouTube não são bloqueados!

Bloquear anúncios no YT é muito difícil porque a Google usa encriptação e servidores gerados aleatoriamente para fornecer os anúncios. Therefore, it's much easier to block ads within the YT app as with an external tool (like Blokada). NewPipe e YouTube Vanced estão cheios de funcionalidades, são óptimas alternativas à aplicação oficial do YouTube, por favor veja qual lhe agrada mais.

Não consigo iniciar o Blokada após a sua instalação.

Existem duas causas para este problema. Em primeiro lugar, verifique se tem uma ligação VPN activa e se a opção VPN sempre ligada está activada. Para o fazer, vá às Definições do seu telemóvel, procure por VPN na secção de Ligações/Rede e pressione na roda dentada ao lado da VPN existente. Desactive essa opção, desconecte-se da ligação VPN activa e tente reiniciar o Blokada.

If you don’t have other VPN connection, check in Blokada if Watchdog is enabled in the Advanced Settings, and if yes, check if you are connected to the internet either with wifi or with mobile data. Blokada won’t start if the Watchdog is enabled and you don’t have an active internet connection.

Se isto não ajudar, reinicie o seu telemóvel, o que também ajuda, por vezes.

O Blokada deixa de funcionar após algum tempo.

O primeiro passo é activar a opção Manter activo no Blokada. Neste caso, o Blokada irá tentar manter-se online, mas por favor note que o sistema do seu dispositivo ainda pode afectar este comportamento. Se tiver activado essa opção e ainda continuar a ver o Blokada a ser interrompido frequentemente, verifique as definições do sistema e desligue qualquer optimização de bateria ou memória referente ao Blokada. Infelizmente, isto é diferente para cada sistema, portanto não podemos indicar onde alterar isto. Verifique as definições do seu dispositivo. Outra causa para este comportamento poderá ser a utilização de uma aplicação para limpar a memória. Neste caso, por favor, inclua o Blokada na lista branca ou lista de exclusões. Se nenhum destes passos ajudou, por favor reporte um erro (veja abaixo como fazê-lo).

Não consigo importar as listas EasyList/AdBlockPlus.

Se tentar importar uma lista (por exemplo, EasyList) que funcione bem com a extensão AdBlock Plus no seu PC ou portátil, o Blokada irá reportar uma mensagem de erro. Isto deve-se ao facto do Blokada utilizar um método diferente no bloqueio de anúncios. Enquanto que o ABP bloqueia e oculta elementos ou partes de um website, o Blokada não pode fazê-lo, uma vez que não é um plugin no navegador; em vez disso, ele redirecciona os acessos ao servidor de anúncios para um endereço específico (0.0.0.0) e, por conseguinte, necessita de uma lista de nomes de endereços IP.

Como funciona o Blokada?

O Blokada é uma fina camada entre a interface de rede (Wi-Fi ou dados móveis) do seu telemóvel e as aplicações que utiliza. To be able to filter every outgoing traffic, it generates a local (no 3rd party service in use) VPN connection and simply blocks any requests towards the hosts defined in the blacklist filters. Este bloqueio é efectuado através de um redireccionamento, o endereço IP do website que aloja o anúncio será traduzido no seu telemóvel para um endereço específico (0.0.0.0), onde nenhum serviço web é executado e, consequentemente, nada será mostrado. Desta forma, o Blokada poupa-lhe tanto no plano de dados móveis como na bateria. Os filtros são actualizados automaticamente e contêm endereços conhecidos por alojarem anúncios, rastreios, spam, malware e outro tipo de conteúdo indesejado ou perigoso. Pode configurar o Blokada para utilizar os seus próprios filtros e listas brancas personalizados, de modo a garantir que o serviço que gostaria de usar não seja bloqueado, sem interrupções.

Posso usar outra VPN enquanto o Blokada está ligado?

Infelizmente, não, é uma limitação do sistema; apenas pode ter uma única ligação VPN activa. Se se ligar a outro serviço, o Blokada irá encerrar.

Quais as opções no Blokada?

Como pode ter observado, o Blokada dispõe de diversas opções no ecrã principal, as quais não foram detalhadas quando iniciou a aplicação pela primeira vez. Por favor, deixe-nos fazer isso agora. Pode observar as seguintes opções:

  • Lista Negra: esta é a lista que contém os endereços a bloquear. Por favor repare que, ao utilizar múltiplas listas. isto pode afectar a velocidade da ligação à Internet e do sistema.
  • Lista Branca: a Lista Branca representa a lista de excepções, onde pode definir os endereços que não deverão ser bloqueados ou as aplicações para as quais o Blokada não filtrará URLs. O Blokada é instalado com diversas aplicações já incluídas na Lista Branca, por defeito.
  • Manter activo: esta opção garante que o Blokada não é pausado ou parado pelo sistema. Por favor, repare que o sistema pode sobrepor-se a esta opção, portanto certifique-se de que as definições do sistema permitem ao Blokada manter-se em execução em segundo plano.
  • Notificações: quando activadas, permitem que seja notificado sempre que um anúncio é bloqueado. Encontra-se activada, por defeito, mas se receber demasiadas notificações, pode desligá-la.
  • Watchdog: when enabled, Blokada will exit if there is no active connection, and start again whenever connection comes back. Quando desactivada, o Blokada irá manter-se permanentemente em execução em segundo plano (mesmo se nenhuma ligação estiver disponível). Por favor, note que esta opção poderá não funcionar em todos os dispositivos.
  • Iniciar no arranque: quando activada, esta opção permite que o Blokada se inicie automaticamente após o arranque do sistema. Por favor, repare que que esta opção envolve configurações ao nível do sistema e, como tal, não é possível garantir que o Blokada seja efectivamente iniciado no arranque. Neste caso, por favor verifique as definições do seu telemóvel. Esta opção está activa por predefinição, se pretender iniciar o Blokada manualmente, desactive esta opção.
  • DNS: se suspeita que alguns sites ou serviços encontram-se bloqueados pelo seu operador, ou que o mesmo recolhe informação sobre a sua actividade na Internet, escolha um DNS seguro a partir da lista. Por defeito, o Blokada utiliza o servidor de DNS do seu sistema.

Opções não configuráveis:

  • Actualizado: se vir isto, significa que está a utilizar a versão mais recente do Blokada. Mantenha a aplicação actualizada, pois desta forma terá menos erros e mais funcionalidades!
  • #Anúncios bloqueados: o número de conteúdo indesejado que o Blokada bloqueou desde que foi iniciado pela primeira vez
  • Endereços: número de servidores que alojam anúncios, malware e outro conteúdo indesejado

O que posso fazer se ainda estiver a ver alguns anúncios?

Alguns anúncios irão surgir sempre, independentemente do bloqueador de anúncios que utilizar em Android. Em particular, anúncios do YouTube ou "pop-ups" de ecrã inteiro são difíceis de bloquear. Se lhe parece que estão a ser deixados passar mais anúncios que anteriormente, tente seguir estes passos:

  • Verifique se o ícone da chave da VPN está presente na barra de estado, no cimo do ecrã. Em caso negativo, inicie o Blokada.
  • Abra o Blokada e verifique se este se encontra activo e se não está a reportar erros.
  • Confirme se o total de endereços bloqueados não é zero. Se for, seleccione alguns filtros da Lista Negra, de forma a bloquear anúncios. If the counter is still zero, grant storage access permission in the System Settings - Applications - Blokada, then clear data and cache and reboot the device.
  • Certifique-se de que não tem demasiadas aplicações na lista branca - talvez seja por isso que está a ver tantos anúncios.
  • Tente resetar os seus filtros (vá ao ecrã da Lista Negra e seleccione Repor Valores Padrão a partir do menu de Acções Rápidas, em cima).
  • Se vir anúncios no seu navegador, desactive qualquer tipo de compressão de dados ou optimização nas definições do navegador.
  • Desactive o Blokada, encerrando-o (deslizando-o para fora no ecrã de "Aplicações Recentes"), e abra-o novamente.
  • Se alterou recentemente as suas definições (filtros, lista banca, etc.), lembre-se que pode demorar algum tempo até que a nova configuração entre em vigor. Aguarde um pouco.
  • Elimine os dados da aplicação do Blokada (nas definições do sistema), ou experimente reinstalá-lo.
  • Reinicie o seu dispositivo.
  • O seu dispositivo pode não ser suportado. Por favor, veja a Base de Dados de Dispositivos.
  • Se nenhum destes passos ajudou, por favor reporte um erro (veja abaixo).

O que é o DNS e qual devo escolher?

O acrónimo "DNS" significa "Domain Name System" (Sistema de Nomes de Domínio) e estes servidores são responsáveis por traduzir os nomes de domínios (por exemplo: omeuexemplo.com) em endereços IP, os quais são utilizados na comunicação entre computadores. A sua operadora tem os seus próprios servidores de DNS; o Blokada está configurado para utilizar esse padrão do sistema. Os computadores nesta lista deverão ser utilizados se suspeitar que o seu ISP recolhe informação sobre a sua actividade na internet, se tem receio sobre riscos de privacidade, ou se alguns sites se encontram bloqueados pelo seu ISP. Não existe uma opção de DNS melhor que as outras, tudo depende das suas preferências e localização e, como tal, se pretende utilizar um servidor de DNS personalizado, por favor leia a respectiva descrição, procure saber mais na internet e leia os comentários sobre os mesmos.

Não consigo instalar uma actualização, o que posso fazer?

Em primeiro lugar, por favor verifique se a versão do Android que possui é a versão 5 ou superior. O Android 4 e versões anteriores não são suportados pelo Blokada 3.0 e suas versões mais recentes. Para confirmar isto, vá às Definições no seu dispositivo, deslize até Acerca do telefone e procure pela Versão Android

Se o seu dispositivo correr o Android 5 ou superior, por favor, reinicie o seu dispositivo. Pode parecer estranho, mas por vezes ajuda. Se isso não tiver resolvido o problema, desinstale o Blokada e volte a instalá-lo. O seu dispositivo pode não ser suportado. Por favor, veja a Base de Dados de Dispositivos.

Se nenhum destes passos ajudou, pro favor adira ao nosso "chat" do Telegram em go.blokada.org/chat para obter ajuda, ou reportar um erro (veja abaixo).

Posso esconder a notificação da VPN?

Infelizmente, não. Alguns dispositivos podem mostrar uma notificação irritante da VPN (um ícone de uma chave), sempre que a VPN estiver activa, não sendo possível ocultá-la. Se lhe for possível, tente actualizar o seu dispositivo para o Android 8 (Oreo), uma vez que versões mais recentes do Android não mostram esta notificação.

Como posso reportar um problema, ideia ou sugestão?

Se estiver a ter problemas, por favor adira ao nosso "chat" do Telegram em go.blokada.org/chat, onde a nossa comunidade ficará feliz por ajudá-lo. Em alternativa, escolha a opção "Comentários" a partir do menu no canto superior direito e siga as instruções. Por favor, forneça tanta informação quanto lhe for possível e indique os passos para reproduzir a ocorrência - isto ajuda muito na resolução do problema! Se for um utilizador pro, pode ir directamente à secção de problemas no GitHub.

O Blokada consegue ver os meus dados privados?

Não. O Blokada não lê nem interfere com a sua informação privada, apenas monitoriza os endereços para os quais o seu dispositivo está a fazer pedidos e bloqueia aqueles que coincidem com os filtros. Sempre que o Blokada bloqueia algo, verá uma notificação (a qual pode ser desactivada). O Blokada não tem acesso ao seu tráfego encriptado.

O Blokada consome muita bateria?

Não. Dependendo das suas definições, o impacto na bateria é mínimo e, na maioria dos dispositivos mais recentes, em particular no Marshmallow (Android 6) ou mais recente, não notará qualquer consumo excessivo de bateria. No entanto, alguns dispositivos menos afortunados poderão sofrer um aumento do consumo de bateria. Por favor, veja a Base de Dados de Dispositivos.

O Blokada irá tornar o meu dispositivo mais lento?

Não. Dependendo das suas definições, o Blokada pode ser bastante eficiente. Na verdade, tanto o desempenho do seu dispositivo como o consumo de bateria podem melhorar significativamente, pois o Blokada previne que o seu dispositivo transfira e processe conteúdo de que não necessita e que não lhe seja relevante. No entanto, alguns utilizadores com dispositivos mais antigos podem sofrer uma quebra no desempenho. Por favor, veja a Base de Dados de Dispositivos para mais detalhes.

O Blokada é realmente gratuito?

Sim! O Blokada começou como um passatempo e continua a ser desenvolvido principalmente por mim. É minha maneira de retribuir para a comunidade. O Blokada é um projecto totalmente de código aberto e permanecerá gratuito eternamente. Dito isto, desenvolvê-lo consome muito tempo, portanto considere apoiar o projecto da forma que lhe for possível (veja abaixo), por favor.
