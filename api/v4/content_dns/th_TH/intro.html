ยินดีต้อนรับและขอบคุณที่ใช้ Blokada DNS!

Blokada เป็นบริการฟรีและโอเพนซอร์สและเคารพความเป็นส่วนตัวของคุณ เมื่อคุณเริ่มใช้แสดงว่าคุณยินยอมตามข้อกำหนดและนโยบายความเป็นส่วนตัวที่คุณสามารถอ่านได้ที่นี้

พวกเราอยากจะช่วยเหลือคุณในขั้นตอนแรกของคุณ:

  1. ค่าเริ่มต้นเซิร์ฟเวอร์ DNS ที่ถูกใช้ในปัจจุบัน คุณสามารถเปลี่ยนแปลงได้โดยการแตะบนไอคอน ค่าเริ่มต้น DNS
  2. Blokada ถูกตั้งให้เริ่มเป็นค่าเริ่มต้นเมื่อโทรศัพท์ของคุณเปิดเครื่อง คุณสามารถปิดมันได้โดยการแตะที่ไอคอน เริ่มตอนบูตเครื่อง
  3. Blokada จะเริ่มทำงานเมื่อคุณเชื่อมต่ออินเทอร์เน็ตเท่านั้น คุณสามารถเปลี่ยนแปลงได้โดยการตั้งค่า ออนไลน์เท่านั้น เป็นปิด
  4. ใน รายการยกเว้น คุณสามารถกำหนดได้ว่าแอพไหนไม่ควรใช้เซิร์ฟเวอร์ DNS ของคุณ พวกเขาจะใช้เซิร์ฟเวอร์ DNS ค่าเริ่มต้นของอุปกรณ์แทน
  5. การแตะด้านบนมุมขวา คุณสามารถค้นหาตัวเลือกอื่นๆ และเมนู ช่วยเหลือ ที่จะช่วยให้คุณได้รู้ข้อมูลรายละเอียดมากขึ้นเกี่ยวกับความแตกต่างของตัวเลือกใน Blokada
