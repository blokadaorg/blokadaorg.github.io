Blokada DNS를 사용해 주셔서 감사합니다!

Blokada는 무료이고 오픈 소스이고 당신의 개인 정보를 관리합니다. 사용을 위한 서비스 약관과 개인 정보 보호 정책을 동의한 내용은 언제든 여기서 확인 가능합니다.

첫 단계를 도와드리고자 합니다:

  1. 현재 기본 DNS 서버가 사용됩니다. 기본 DNS 아이콘을 눌러 변경할 수 있습니다.
  2. Blokada 는 휴대폰이 작동될 때 시작하도록 설정되어 있습니다. 부팅 시 시작 을 눌러 끌 수 있습니다.
  3. Blokada는 인터넷에 연결되어있을 때만 실행됩니다. 온라인 전용 끄기로 설정하여 변경할 수 있습니다.
  4. 허용 목록 에서는 DNS 서버를 사용하지 않아야하는 앱을 정의 할 수 있습니다. 대신 기본 장치 DNS 서버를 사용합니다.
  5. 오른쪽 상단 모서리를 탭하면 더 많은 옵션과 Blokada의 다양한 옵션에 대한 자세한 정보를 얻을 수있는 도움말 메뉴를 찾을 수 있습니다.
