ようこそ、Blokada の DNS を使用していただきありがとうございます!

Blokada はフリーでオープンソースであり、あなたのプライバシーを尊重します。 By using it you agree to the Terms of Service and Privacy Policy, which you can always find here.

We would like to assist you with your first steps:

  1. Default DNS servers are used currently. You can change it by tapping on Default DNS icon.
  2. Blokada は、あなたの携帯電話が起動したときに開始に設定されます。 You can turn if off by tapping on Start on boot.
  3. Blokada は、インターネットに接続している場合にのみ実行されます。 オフにする 通知 を設定することによって変更できます。
  4. In the Whitelist you can define which apps shouldn't use your DNS servers. They will use default device DNS servers instead.
  5. 右上隅をタップすることで多くのオプションと Blokada のさまざまなオプションに関する詳細情報を得ることができて[ヘルプ] メニューを検索できます。
